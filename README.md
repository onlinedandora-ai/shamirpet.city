# shamirpet.city

Local news, business directory, and community portal for Shamirpet. Built with React, TypeScript, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Development

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
```

## Tech Stack

- **Framework**: React + Vite + TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Linting**: Oxlint
